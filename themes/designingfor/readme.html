Quick-start Guide

Features of the Theme

  1. A dynamic jQuery slide show using the popular jQuery Cycle Plugin
  2. Widgetized sidebar with access to the Custom Menus in the newest version of WordPress.
  3. Dynamically generated thumbnails based on the images within the posts. No need to manually setup your article thumbnails. The theme takes care of it for you.
  4. A clean design that will not get in the way of your great content.

Installing the Theme

  1. Download the Theme archive and extract the files it contains. You may need to preserve the directory structure in the archive when extracting these files. Follow the guidelines provided by your Theme author.
  2. Using an FTP client to access your host web server, create a directory to contain your Theme in the wp-content/themes directory provided by WordPress. For example, a Theme named Test should be in wp-content/themes/test. Your Theme may provide this directory as part of the archive.
  3. Upload the Theme files to the new directory on your host server.
  4. Follow the instructions below for selecting the new Theme.

Selecting the Active Theme

  1. Log in to the WordPress Administration Panel.
  2. Select the Appearance panel, then Themes.
  3. From the Available Themes section, click on the Theme title (or Theme screenshot) for the Theme you wish to activate.
  4. A preview of the Theme will be shown.
  5. To activate the Theme click the Activate “Theme Name” link in the top right.

Frequently Asked Questions

Q: How do I change the speed of the slide show on the homepage?
A: The script necessary to run the slide show is located in the header.php file within the theme folder.

$('#slideshow').cycle({
fx: 'fade',
speed: '750',
timeout: 5000,
next: '#next',
prev: '#prev'
});

If you would like to increase / decrease the time between transitions simply increase / decrease the value of timeout. The time is in milliseconds, so for 10 seconds we would increase the value to 10000 and 3 seconds would be 3000. To find out more information about the jQuery Cycle Plugin visit their website.

Q: How do I show more than 3 slides in the slide show?
A: You can increase the number of posts pulled into the slide show by opening up your home.php file located within the theme folder and look for this line in the code:

<?php $featured = new WP_Query(); $featured->query('showposts=3'); while ($featured->have_posts()) : $featured->the_post(); ?>

You’ll want to change the value of showposts to increase the number of slides / posts shown in the slide show. For example, if we wanted to show 5 instead of 3 we would change showposts=3 to showposts=5.

Q: How do I change the logo?
A: The easiest way to change the logo is to overwrite the current logo.gif image located in the /images/ folder of the theme’s folder. Note: You may need to perform some anti-aliasing touch-up work depending on the quality & color of the aliasing of your .gif. Using a .gif will allow for transparency when applying the image over the striped background.

If you have experience using .png (and are not worried about older users of Internet Explorer) you might want to try a .png instead since these give semi-transparency and can ultimately lead to much nicer integration. When using a .png you will need to modify the following line of code in the header.php file of the theme:

<div id="logo"><a href="<?php bloginfo('home'); ?>"><img src="<?php bloginfo('stylesheet_directory'); ?>/images/logo.gif" alt="Designing for the Masses" /></a></div>

Change the logo.gif to whatever you’ve named your .png image.
